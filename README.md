XTEROID: Intelligent Performance Engine

XTEROID is a sophisticated system service and performance engine designed specifically for Android devices powered by the MediaTek Dimensity 8050 (MT6893Z) and similar chipsets. Unlike static performance tweaks, XTEROID operates as a dynamic background daemon that monitors active applications in real-time to balance raw power with thermal efficiency.


Core Functionality
The engine functions through a tiered approach to system management:


• Dynamic Scaling: Automatically adjusts CPU and GPU frequencies based on whether the user is gaming, using a launcher, or performing low-impact tasks.

• Context Awareness: Utilizing a local database from the Transsion Smartpanel, it identifies "Game Space" apps to trigger high-performance profiles like Esports Mode.

• Thermal Intelligence: Features a "Retemperate" function that manages battery temperature reporting and can disable system throttling when maximum performance is required.

• Display Optimization: Synchronizes the device refresh rate (60Hz, 90Hz, 120Hz) with the active performance profile to ensure a fluid visual experience.

• System Debloating: Integrated initialization scripts disable aggressive GMS telemetry and analytics to reduce background CPU overhead.
